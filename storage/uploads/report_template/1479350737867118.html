Оны шилдэг ажилтан урамшуулах тухай

   editable( ) үндэслэн ТУШААХ нь:
  1.   #year# онд ажил үйлсээрээ хошуучлан манлайлж, үр бүтээлтэй ажилласан нэр бүхий Хавсралтад заасан ажилтнуудыг  доорхи шагнал урамшууллар шагнасугай. Үүнд: 
  • Оны шилдэг ажилтан - Өргөмжлөл,    editable( )
  • МЗХолбооны шагнал- Холбооны үнэмлэх, медаль,   editable( )
  • Засгийн газрын хэрэгжүүлэгч агентлаг Гэр бүл, хүүхэд, залуучуудын хөгжлийн газар- Холбооны үнэмлэх, медаль, editable( )
  2.   Дээрх заалттай холбогдуулан өргөмжлөл, мөнгөн урамшууллын зардлыг холбогдох данснаас гаргахыг Ахлах Нягтлан бодогч Э.Сайнзаяад даалгасугай.
  3.   Тушаалын биелэлтэд хяналт тавьж ажиллахыг Хүний нөөцийн менежер Б.Хосчимэгт үүрэг болгосугай.

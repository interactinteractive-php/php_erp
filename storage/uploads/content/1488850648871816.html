SQL NULL Values

    
    Энэ нь хоосон буюу ямар ч утгагүй гэсэн санааг илтгэдэг. Өгөгдлийн сангийн хүснэгтийн тухайн нэг талбар  NULL утгатай байна гэдэг нь тухайн талбар ямар ч утгагүй хоосон байна гэсэн үг юм. Хэрэв тухайн талбарт өгөгдөл нэмэх эсвэл утгыг засахгүйгээр хоосон орхиж хадгалах үед энэ талбар нь Null утгатайгаар өгөгдлийн санд хадгалагддаг. 

Хэрэглээ: Утгын дээрх төрөл нь өгөгдлийн сангийн чухал ойлголтуудын нэг. Бид 0 утгаас ялгаатай гэдгийг ойлгох нь чухал. Тухайн хүснэгтийн аль нэг талбар хоосон эсвэл утгатай байгаа эсэхийг шүүж олох боломжтой байдаг. 

Бичигдэх хэлбэр /syntax/:
SELECT column_name1, column_name2 ... FROM table_name WHERE column_name IS NULL / IS NOT NULL;  
 
Зураг 01 -с жишээнд ашиглах хүснэгтийг харна уу.

    
                                                                                                                                                             Зураг 01 Null шалгах жишээнд ашиглах Persons хүснэгт    
   
Зурагт Persons хүснэгтийн Address багана нь сонголтын буюу хоосон утгатай байж болохоор үүсгэгдсэн байгааг харуулсан байна. Өөрөөр хэлбэл утга оруулахгүй орхивол автоматаар Null утгатай болох юм.

Жишээ-1: Дээрх Persons хүснэгтээс Address баганы утга хоосон байгааг шалгах буюу хаягийн мэдээлэл оруулаагүй хүний мэдээллийг харах шаардлага гарсан гэж үзье.

SELECT LastName, FirstName, Address FROM Persons WHERE Address IS NULL;

Зураг 02 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.
    
                                                                                                                        Зураг 02 Хаягийн бүртгэлгүй буюу NULL утгатай  байгаа хүмүүсийн мэдээлэл    
    Зурагт IS NULL операторын хэрэглээг харуулсан бөгөөд Persons хүснэгтээс Address баганы утга хоосон байгаа бичлэгүүдийг select хийж байна. Эсрэг тохиолдолд буюу хаягийн бүртгэлтэй хүмүүсийн мэдээллийг харахыг хүсвэл IS NOT NULL операторыг ашиглах юм.

Text Editor гэж юу вэ?

    
   

Text Editor гэдэг нь Field метадатаны төрөл юм. Text Editor төрөлтэй Field нь html content буюу тухайн html - н элементүүдээс бүрдэх их хэмжээний текст агуулсан хуудас бүхий утга авдаг талбар юм. Энэ нь текстийн өнгө, хэлбэр, загварыг тодорхойлохойс гадна зураг, файл оруулах, хүснэгт бэлтгэх зэрэг маш олон боломжуудтай юм. Text Editor нь үндсэндээ хэрэгсэл, утга оруулах талбар гэсэн 2 хэсгээс бүрддэг бөгөөд хэрэглэгч оруулсан текст бичвэрийнхээ харагдацыг тодорхой хэрэгсэлүүдийн тусламжтай тохируулдаг. Мөн html, css код бичиж контент үүсгэх боломжтой юм.

    
    Text Editor-г үүсгэхдээ Enterprice innovation studio хэсэгт нэмэх товчны тусламжтайгаар мета нэмэх ба баруун талд байрлах төрөл хэсгээс Field-ийг сонгосны дараа өгөгдлийн төрөл хэсгээс Text Editor-ийг сонгож үүсгэнэ.

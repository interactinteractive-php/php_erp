SQL UNION Operators

    
     Энэ нь SQL хэлний select query ажиллуулсаны үр дүн үүсэх 2 болон түүнээс дээш хүснэгтүүдийг нэгтгэж харах боломж олгодог SQL хэлний operator юм. UNION  нь дараах зарчимтай.
  • Select statement -р үүсэх хүснэгт бүр нь ижил тооны баганы тоотой байх
  • Баганууд нь ижил өгөгдлийн төрөлтэй байх
  • Select statement -р үүсэх хүснэгт бүрийн баганы дараалал ижилхэн байх
Хэрэглээ: Тухайн олон хүснэгтүүдээр тайлан үүсгэх, өгөгдлийн давхардал арилгаж харах, бүх бичлэгийн тоог олох мөн бүх бичлэгүүдийг нэгтгэж нэгдсэн хайлт хийж, тооцоолол гаргах зэрэг маш өрөн хүрээнд хэрэглэнэ. Мөн Union -г хоёр төрлөөр хэрэгжүүлдэг. Эдгээр нь Union болон Union All юм. 
UNION: Энэ нь нэгтгэж байгаа хүснэгтүүдийн баганы бичлэгүүдийг Distict хийж өгөгдлийн давхардалгүй харуулдаг.
UNION ALL: Энэ нь нэгтгэж байгаа хүснэгтүүдийн баганы бүх бичлэгүүдийг давхардсан утгаар харуулдаг.

Бичигдэх хэлбэр /syntax/:
UNION
SELECT column_name(s) FROM table1 UNION SELECT column_name(s) FROM table2;

UNION ALL
SELECT column_name(s) FROM table1 UNION ALL SELECT column_name(s) FROM table2;
 
Зураг 01 -с дээрх жишээнд ашиглагдах Customers хүснэгтийн City болон Country баганы бичлэгүүдийг харна уу.

    
     
                                                                                                                                                                          Зураг 01 Customers хүснэгтийн City, Country баганы бичлэгүүд		    
    Зураг 02 -с жишээнд ашиглагдах  Suppliers хүснэгтийн City болон Country баганы бичлэгүүдүүдийг харна уу    
   
                                                                                                                                                                           Зураг 02 Suppliers хүснэгтийн City, Country баганы бичлэгүүд		    
   
Жишээ-1: UNION operator ашиглан дээрх 2 үр дүнгийн хүснэгтүүдийг City талбараар нь нэгтгэж харцгаая.

SELECT City FROM Customers UNION SELECT City FROM Suppliers;

Зураг 03 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.

    
   
                                                                                                        Зураг 03 Union ашиглан Customers болон Suppliers хүснэгтүүдийн City талбарыг нэгтгэж харуулж байна.		    
   
Зурагт UNION операторын хэрэглээг харуулсан бөгөөд Customers болон Suppliers хүснэгтүүдийн City баганыг нэгтэж харуулахдаа давхардсан утгыг арилган харуулсан байгаа нь бичлэгийн тооноос харагдаж байна.

Жишээ-2: UNION ALL operator ашиглан дээрх 2 select statement -н үр дүнгийн хүснэгтүүдийг өгөгдлийн давхардал арилгахгүйгээр, шүүлтүүр тавьж нэгтгэж харцгаая.

SELECT City, Country FROM Customers
WHERE Country='Germany'
UNION ALL
SELECT City, Country FROM Suppliers
WHERE Country='Germany';

Зураг 04 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.
    
   

                                  Зураг 04 Зураг 04 Union All ашиглан Customers болон Suppliers хүснэгтүүдийн City, Country талбаруудын бичлэгүүдийг шүүлт тавьж нэгтгэж байна.
    
   
Зурагт UNION ALL операторын хэрэглээг харуулсан бөгөөд Customers болон Suppliers хүнэгтүүдийн Country баганы утга 'Germany' -тай байх үед City болон Country багануудын бичлэгүүдийг нэгтгэж харуулсан байна.

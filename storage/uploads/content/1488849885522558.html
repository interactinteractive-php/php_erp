SQL WILDCARDS

    
    Энэ нь өгөгдлийн сангийн хүснэгтүүдээс төсөөтэй утгаар хайлт хийх боломжийг олгодог LIKE операторт хэрэглэгддэг нэг болон түүнээс дээш тэмдэгтүүдийг орлуулж ашигладаг тодорхой тооны тэмдэгтүүд юм.

Хэрэглээ: Ерөнхийдөө хоёр төрлийн тэмдэгт нь Like операторыг хэрэгжүүлэхэд ашиглагдана.
  • % - Хувийн тэмдэг нь 0 утгыг илтгэдэг бөгөөд нэг эсвэл олон тэмдэгтээр төсөөтэй эсэхийг шалгахад хэрэглэгдэнэ.
  • _ - Доогуур зураас тэмдэгт нь дан тэмдэгтийг илтгэдэг
Түүнчлэн дээрх хоёр төрлийн хэрэгжүүлэлтийг хослуулж ашиглах боломжтой бөгөөд AND, OR оператуудтай зохицон ажилладаг. Зарим нэг хэрэглээг доор хүснэгтэд харууллаа.
LIKE Оператор Тайлбар
WHERE CustomerName LIKE 'a%' Хэрэглэгчийн нэр "a" тэмдэгтээр эхэлсэн бол биелнэ.
WHERE CustomerName LIKE '%a' Хэрэглэгчийн нэр "a" тэмдэгтээр төгссөн бол биелнэ.
WHERE CustomerName LIKE '%or%' Хэрэглэгчийн нэр аль ч хэсэгтээ "or" тэмдэгт мөрийг агуулсан бол биелнэ.
WHERE CustomerName LIKE '_r%' Хэрэглэгчийн нэрийн эхлэл хэсгийн 2 дах тэмдэгт нь "r" байх бол биелнэ.
WHERE CustomerName LIKE 'a_%_%' Хэрэглэгчийн нэр "a" тэмдэгтээр эхэлсэн бөгөөд ядаж 3 тэмдэгтийн урттай бол биелнэ.
WHERE ContactName LIKE 'a%o' Хэрэглэгчийн нэр "a" тэмдэгтээр эхлээд "о" тэмдэгтээр төгссөн бол биелнэ.

 
Жишээ-1: Хэрэглэгчийн мэдээллийн жагсаалтаас Antonio нэртэй хэрэглэгчийн мэдээллийг харах гэтэл бүтэн нэрийг мэдэхгүй байсан гэж үзье. Энэ үед хэрэглэгчийн нэрний эхлэл хэсгийг санаж байхад LIKE операторийг ашиглах нь тохиромжтой. 

SELECT * FROM Customers; WHERE CustomerName LIKE 'an%';

Зураг 01 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.

    
   
                                                                                                                                        Зураг 01 LIKE ашиглан 'an' тэмдэгт мөрөөр эхэлсэн нэртэй хэрэглэгчдийн мэдээлэл		    
   
Зурагт % тэмдэгтийг Like операторын хэрэгжүүлэлтэнд ашигласан үр дүнг харуулсан бөгөөд тус тэмдэгтээр хайх утгын хойд хэсгийг бүхэлд нь төлөөлүүлэн авсан байгаа нь харагдаж байна. Өөрөөр хэлбэл % тэмдэгтийг үгийн адагт оруулан бичвэл зөвхөн эхлэл хэсгийн утга таарч байх үеийн бичлэгүүдийг харуулна гэсэн үг юм.

Жишээ-02: Тэмдэгтүүдийн жагсаалтаар хэрхэн LIKE операторыг хэрэгжүүлэхийг дараах query ажилласан үр дүнгээс харцгаая.

SELECT * FROM Customers WHERE City LIKE '[a-c]%';

Зураг 02 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.

    
    Зураг 02 '[a-c]' тэмдэгтүүдээр эхэлсэн хотын хэрэглэгчдийн мэдээлэл    
   
Зурагт тэмдэгтүүдийн жагсаалтаар эхлэлийн тэмдэгтийг тохируулан шүүлт хийсэн үр дүнг харуулсан бөгөөд 'a', 'b', 'c' тэмдэгтүүдээр эхэлсэн хотын хэрэглэгчдийн мэдээллийг жагсаасан байна.   

Жишээ-03: '_'  тэмдэгтээр хэрхэн LIKE операторыг хэрэгжүүлэхийг дараах query ажилласан үр дүнгээс харцгаая. 

SELECT * FROM Customers WHERE City LIKE 'L_n_on';

Зураг 03 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.

    
    Зураг 03 'L'-р эхэлсэн дундаа 'o' тэмдэгт агуулсан 'on'-р төгссөн хотын хэрэглэгчдийн мэдээлэл    
    Зурат дор хаяж 3 тэмдэгтээс бүрдсэн, 'L' тэмдэгтээр эхэлсэн, 'O' тэмдэгт дундаа агуулсан, 'ON' тэмдэгтүүдээр төгссөн хотын хэрэглэгчдийн мэдээллийг харуулсан байна. Энд '_' тэмдэгт нь нэг тэмдэгтийг илэрхийлж байгаа нь харагдаж байна.

SQL COUNT(), AVG(), SUM()

    
    Эдгээр нь SQL хэлний Aggregate functions ангилалд багтдаг фунцкууд юм. 

  • COUNT() нь шүүлтүүрээр гарч ирэх нийт үр дүн буюу хүснэгтийн нийт бичлэгийн тоог буцаана.
  • AVG() нь тоон төрлийн утгатай баганын утгуудын дундаж утгыг тооцоолон буцаана.
  • SUM() нь тоон төрлийн утгатай баганын утгуудын нийлбэр утгыг тооцоолон буцаана.
Хэрэглээ: Дээрх 3 төрлийн функцуудыг тооцоолол хийхэд маш өргөнөөр ашигладаг.

Бичигдэх хэлбэр /syntax/:
COUNT()
SELECT COUNT(column_name) FROM table_name WHERE condition;

AVG()
SELECT AVG(column_name) FROM table_name WHERE condition;;

SUM()
SELECT SUM(column_name) FROM table_name WHERE condition;
 
Зураг 01 -с жишээнд ашиглагдах  Producs хүснэгтийг харна уу.

    
     
                                                                                                                                                                                                     Зураг 01 Products хүснэгт		    
    Жишээ-1: Өгөгдлийн сангийн бүтээгдэхүүн бюу Products хүснэгтэд нийт хэдэн бүтээгдэхүүн байгааг тоолж харах шаардлагатай болсон гэж үзээд дараах query -г бичиж, ямар үр дүн гарч ирэхийг харцгаая.

SELECT COUNT(ProductID) AS CountOfProducs FROM Products;

Зураг 02 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.

    
                                                                                                                                                                                                   Зураг 02 Product хүснэгтийн нийт бичлэгийн тоо    
   
Зурагт Products хүснэгтийн бичлэгийн тоог Count функц ашиглан alians хийж харуулсан бөгөөд баганы нэр заахгүйгээр Count(*) хэлбэрээр бичиж болно.

Жишээ-2: Products хүснэгтийн нийт бүтээгдэхүүнүүдийн дундаж үнийг дараах query -р олж харцгаая.

SELECT AVG(Price) AS AvgPriceOfProducs FROM Products;

Зурагт Product хүснэгтийн Price баганы дундаж үнийг AVG() функц ашиглан олж Alians хийж харуулсан байна.

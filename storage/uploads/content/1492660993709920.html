SQL ANY, ALL Operators

    
    Эдгээр нь SQL хэлний өгөгдлийн сангийн хүснэгтүүдээс тодорхой нөхцөл, шалгуурын дагуу шүүлт хийхэд ашиглагддаг WHERE мөн Having clause -тай хамт ажилладаг operator - ууд юм. 

  • ANY оператор нь тодорхойлсон нөхцөл subquery -н үр дүнтэй буюу аль нэг утгатай таарч байх үед биелэнэ
  • ALL оператор нь тодорхойлсон нөхцөл subquery -н бүх үр дүнтэй буюу  таарч байх үед биелэнэ.
Хэрэглээ: Өгөгдлийн сангийн хүснэгтүүдээс шүүлт хийж харахдаа subquery ажилласан үр дүнгээс хайлт хийж таарах утгуудыг буцаадаг онцлогтой операторууд юм. Any болон All нь бичигдэх хэлбэрийнхээ хувьд (=, <>, !=, >, >=, <, or <=) зэрэг харьцуулалтын операторуудаас авч хэрэглэгдэг.

Бичигдэх хэлбэр /syntax/:
ANY
SELECT column_name(s). FROM table_name WHERE column_name operator ANY (SELECT column_name FROM table_name WHERE condition);

ALL
SELECT column_name(s). FROM table_name WHERE column_name operator ALL (SELECT column_name FROM table_name WHERE condition);


Зураг 01 -с жишээнд ашиглагдах Products хүснэгтийг харна уу.

    
     
                                                                                                                                                                                                                                Зураг 01 Products хүснэгт		    
    Зураг 02 -с жишээнд ашиглагдах OrderDetails хүснэгтийг харна уу.
    
   
                                                                                                                                                                                                                           Зураг 02 OrderDetails хүснэгт		    
   
Жишээ-1: ANY оператор ашигласан дараах query -г хэрхэн ажиллаж ямар үр дүн буцаахыг харцгаая. Гол зарчим нь subquery -р гарч ирэх үр дүнгийн жагсаалтын утгатай нөхцөл таарч байх үеийн бичлэгүүдийг буцаах ёстой юм.

SELECT ProductName FROM Products WHERE ProductID = ANY  (SELECT ProductID FROM OrderDetails WHERE Quantity = 10);

Зураг 03 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.

    
   
                                                                                                                                                            Зураг 03 ANY оператор ашиглан Products хүснэгтээс шүүлт хийж байна		    
   
Зурагт ANY оператор ашиглан шүүлт хийж харуулсан бөгөөд OrderDetails хүснэгтээс Quantity баганы утга 10 -тай тэнцүү байх үеийн бүтээгдэхүүний дугаарын  жагсаалттай Products  хүснэгтийн ProductID баганы утгуудаас тохирч үед ProductName буюу бүтээгдэхүүний нэр талбарын утгуудыг жагсааж харуулсан байгаа юм.

Жишээ-2: ALL оператор ашиглан дараах query -г хэрхэн ажиллаж ямар үр дүн буцаахыг харцгаая. Гол зарчим нь subquery -р гарч ирэх жагсаалтын харгалзах утгуудтай нөхцөлд шалгах баганы утга нэгэн зэрэг таарч байх ёстой юм.

SELECT ProductName FROM Products WHERE ProductID = ALL  (SELECT ProductID FROM OrderDetails WHERE Quantity = 10);

Зураг 04 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.
    
   

                                                                                                                                                            Зураг 04 ALL оператор ашиглан Products хүснэгтээс шүүлт хийж байна.
    
   
Зурагт ALL оператор ашиглан шүүлт хийж харуулсан бөгөөд үр дүн олдоогүй байна. Энд нөхцөлд тодорхйосон багану утга subquery -р гарч ирэх жагсаалтын харгалзах утгуудтай нэгэн зэрэг тэнцүү байх ёстой байсан учир үр дүнд ямар ч бичлэг олдоогүй юм.

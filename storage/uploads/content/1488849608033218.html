SQL AND, OR, NOT Operators

    
    Эдгээр нь SQL хэлний өгөгдлийн сангийн хүснэгтүүдээс тодорхой нөхцөл, шалгуурын дагуу шүүлт хийхэд ашиглагддаг WHERE clause -тай хамт ажилладаг логик operator - ууд юм. AND болон OR операторууд нь 1 -с олон нөхцөлөөр шүүлт хийх тохиолдолд хэрэглэгддэг. Мөн AND, OR, NOT операторууд өөр хоорондоо зохицон ажиллах бүрэн боломжтой байдаг. 

  • AND оператор нь тодорхойлсон нөхцөлүүд бүгд биелэж байх үед ажиллана.
  • OR оператор нь тодорхойлсон нөхцөлүүдээс аль нэг нь биелэж байх үед ажиллана.
  • NOT оператор нь нөхцөл үнэн биш худал байх үед ажиллана.
Хэрэглээ: Өгөгдлийн сангийн хүснэгтүүдтэй харьцан ажиллах үед өөрийн хүссэн мэдээллээ шүүлт хийж харахдаа нэгээс олон нөхцөл, шалгуураар яг тохирох үр дүнгээ гаргаж авах шаардлага тулгардаг. Энэ тохиолдолд дээрх 3 төрлийн операторуудыг хэрэглэнэ.

Бичигдэх хэлбэр /syntax/:
AND
SELECT column1, column2, ... FROM table_name WHERE condition1 AND condition2 AND condition3 ...;
OR
SELECT column1, column2, ... FROM table_name WHERE condition1 OR condition2 OR condition3 ...;
NOT
SELECT column1, column2, ... FROM table_name WHERE NOT condition;
 
Жишээ-1: Хэрэглэгчдийн мэдээллийн жагсаалтаас Герман улсын Берлин хотод амьдардаг хэрэглэгчийн мэдээллийг харуулах шаардлага гарсан гэж үзье.
SELECT * FROM Customers;

Зураг 01 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.		    
                                                                                                                                                                                                                                 Зураг 01 хэрэглэгчдийн жагсаалт    
    Зурагт Customers хүснэгтийн мэдээллийг SELECT query ашиглан харуулсан байна. Энд * тэмдэгт нь тухайн хүснэгтийн бүх талбарын select хийж байгааг илэрхийлнэ.

SELECT * FROM Customers WHERE Country='Germany' AND City='Berlin';

Зураг 02 -с дээрх AND ашигласан SELECT query хэрхэн ажилласан байгааг харна уу.
    
   
                                                                                                                                                                       Зураг 02 Германы Берлин хотод амьдардаг хэрэглэгчийн мэдээлэл		    
    Зурагт AND операторын хэрэглээг харуулсан бөгөөд Customers хүснэгтээс Country баганы утга 'Germany' -тэй болон City баганы утга 'Berlin' -тэй тэнцүү байх үейин бичлэгийг бүх талбараар select хийж харуулсан байна. Энд тэмдэгт мөр /varchar/ төрлийн өгөгдлийг нөхцөл болгон бичихдээ дан хашилтан дунд буюу 'data' байдлаар бичнэ.

Жишээ-2: Хэрэглэгчдийн жагсаалтаас 'Ana Trujillo' болон 'Hanna Moos' нэртэй хэрэглэгчдийн мэдээллийг харах шаардлагатай болсон гэж үзье.

SELECT * FROM Customers WHERE ContactName='Ana Trujillo' OR ContactName='Hanna Moos';

Зурагт OR операторын хэрэглээг харуулсан бөгөөд Customers хүснэгтээс ContanctName баганы утга 'Ana Trujillo' эсвэл 'Hanna Moos' байх үеийн бичлэгүүдийг бүх талбараар select хийж харуулсан байна.

Жишээ-3: Хэрэглэгчдийн жагсаалтаас 'Germany' -с бусад орны хэрэглэгчдийн мэдээллийг харах шаардлага гарсан гэж үзье.

SELECT * FROM Customers WHERE NOT Country='Germany';

Зураг 04 -с дээрх NOT ашигласан SELECT query хэрхэн ажилласан байгааг харна уу.
    
   

                                                                                                                                                                                     Зураг 04 Германаас бусад орны хэрэглэгчдийн мэдээлэл
    
   
Зурагт NOT операторын хэрэглээг харуулсан бөгөөд Customers хүснэгтээс Country баганы утга 'Germany' биш байх үеийн бичлэгүүдийг бүх талбараар select хийж харуулсан байна.

Жишээ-4: Customers хүснэгтэд AND, OR операторуудыг ашиглан шүүлт хийх шаардлага гарсан гэж үзье.

SELECT * FROM Customers WHERE Country='Germany' AND (City='Berlin' OR City='Mannheim');

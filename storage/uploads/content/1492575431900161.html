SQL MIN() And MAX() Functions

    
    Эдгээр функц нь тухайн хүснэгтийн сонгогдсон баганы хамгийн бага болон хамгийн их утгыг буцаадаг SQL хэлний functions юм. 

Хэрэглээ: MIN() болон MAX() функцуудыг тооцоолох үйлдэлд мөн шүүлтүүртэй хамт хэрэглэж, хүссэн үр дүнгээ гаргаж авах зэрэгт ихээр ашиглана. 

Бичигдэх хэлбэр /syntax/:
MIN() 
SELECT MIN(column_name) FROM table_name WHERE condition;
MAX()
SELECT MAX(column_name) FROM table_name WHERE condition;

Зураг 01 -с жишээнд ашиглагдах Products хүснэгтийг харна уу.

    
                                                                                                                                                                                                       Зураг 01 Products хүснэгт    
   
Жишээ-1: Products хүснэгтийн Price баганы хамгийн бага утгыг MIN() функц ашиглан олцгоосё.

SELECT MIN(Price) AS SmallestPrice FROM Products;

Зураг 02 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.

    
   
                                                                                                                    Зураг 02 Products хүснэгтийн Price баганы хамгийн бага утга.		    
   
Зурагт MIN() функц ашигласны дараах үр дүнг харуулсан Products хүснэгтийн Price баганы утгуудаас хамгийн бага утгыг буцаасан байна. 

Жишээ-2: Products хүснэгтийн Price баганы хамгийн их утгыг MAX() функц ашиглан олцгоосё.

SELECT MAX(Price) AS LargestPrice FROM Products;

Зураг 03 Products хүснэгтийн Price баганы хамгийн их утга.		    
   
Зурагт MAX() функц ашигласны дараах үр дүнг харуулсан Products хүснэгтийн Price баганы утгуудаас хамгийн их утгыг буцаасан байна.

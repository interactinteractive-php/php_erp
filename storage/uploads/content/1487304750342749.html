Statement гэж юу вэ?

    
    Statement нь Veritech ERP | Enterprice innovation studio дээр тайлангийн загвар боловсруулж хэвлэх болон экспортлох зэрэг олон боломжуудаар хангагдсан бүрэн цогц бие даасан байдлаар ажиллах нэг төрлийн метадата юм. Тус метадата нь DataView буюу жагсаалтыг оролтын параметр болгож авдаг. Түүн дээрээ үндэслэн загвар тодорхойлдог.

Statement үүсгэхдээ Enterprice innovation studio хэсэгт байрлах нэмэх товчны тусламжтайгаар мета нэмэх ба баруун талд байрлах төрөл хэсгээс Statement -г сонгож, үндсэн тохиргоог тохируулж үүсгэнэ.

    Зураг 02 Statement -н үндсэн тохиргоо    
                       
                                                    Зураг 02 Statement -н үндсэн тохиргоо
    
   
Дээрхи зурагт харуулсан тохиргоонуудаас гол ашиглагдах тохиргоодын тайлбарыг дараах жагсаалтаас харна уу.
  • Theme - WorkSpace үүсгэх олон төрлийн загваруудаас өөрийн хэрэгцээнд тохируулан сонгох тохиргоо. Сонгогдсон загварын ямар бүтцийн дагуу хэрхэн харагдахыг тус тохиргооны ард байрлах 3 цэгээр илэрхийлсэн товч дээр дарж харна.
  • DataView - Тухайн WorkSpace -н аль нэг дэд цонхонд эсвэл ерөнхий загварын тодорхой хэсэгт тогтмол мэдээллүүд оруулж өгдөг. Энэ үед тэдгээр мэдээллүүдийг DataView дээр тохируулсан жагсаалтын багануудаас сонгон авч тохируулдаг. Өөрөөр хэлбэл WorkSpace -т ашиглах суурь жагсаалтыг тохируулна.
  • Root Menu - Тухайн сонгосон загварын үндсэн меню байрлах хэсэгт меню метадата тохируулах тохиргоо.  
  • Default Menu - WorkSpace дуудагдаж ажиллах үед хамгийн түрүүнд харагдах менюг сонгох тохиргоо.
  • Байршилын тохиргоо - WorkSpace дээр тогтмол мэдээллүүд харуулах бол харгалзах байрлал  тохируулах тохиргоо юм.
  • Param map - WorkSpace -н дэд цонхонд олон төрлийн бизнес процессууд байрлаж ажиллах шаардлагатай үед тохируулах бөгөөд процессын параметр тохируулага хийдэг. Хэрэв тус тохиргоог тохируулаагүй бол бизнес процессууд ажиллахгүй алдаа гардаг.

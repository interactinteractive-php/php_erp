Expression тохируулах
Бичигдэх хэлбэр, өгөгдөл урсгал, функцын ажиллагаа бүгд JavaScript хэлний зарчимд тулгуурлаж явах бөгөөд дээрхи хүснэгтүүдэд тодорхойлсон төрөл, функцын тусламжтай Veritech ERP - тай бүрэн зохицож ажиллана. 		    
   
Зураг 01 Expression тохируулах үндсэн цонх
    
   
                                                                                                                                                                                                 Зураг 01 Expression тохируулах үндсэн цонх		    
   
Дээрхи зурагт бизнес процессын оролтын параметрүүдийг ашиглан expression тохируулах ерөнхий цонхыг харуулсан байна. Expression нь Path /1/ -р дамжуулан утга оруулах талбарт хандаж, тэдгээрийн утга, харагдац зэргийг өөрчилж, үйл ажиллагааг өргөжүүлдэг. Path нь оролтын параметрийн id бөгөөд HTML элемент болж үүсэхдээ datapath гэх attribute -н утга болон ордог. Ингэснээр [firstName], [lastName] зэрэг path -тай талбаруудад хандах боломжтой болдог юм.
1 - Тухайн процесст ашиглагдаж буй оролтын параметрүүдын жагсаалтыг харуулах хэсэг.
2 - expression бичих төрөл буюу харгалзах tab -г сонгох хэсэг.
3 - expression javascript -н code оруулах хэсэг.		    
   
Жишээ 01: Ажилтан нэмэх үед регистрийн дугаарыг оруулахад нас, төрсөн огноо зэргийг тооцоолж харуулдаг байхаар Expression бичиж тохируулъя. 

Зураг 02 Expression тохируулах цонх руу шилжих		    
   

                                                                                                                                                      Зураг 02 Expression тохируулах цонх руу шилжих		    
   
Дээрхи зурагт ажилтан нэмэх процессын Expression тохируулах цонх руу шилжих 2 төрлийн алхамыг харуулсан байна.

Зураг 03 Ажилтан нэмэх процессын Expression тохируулах
    
   

                                                                                                                                                              Зураг 03 Ажилтан нэмэх процессын Expression тохируулах		    
   
1 - With Event -д registgerNumber гэсэн path id -тай утга оруулах талбарт товч дарах буюу шинээр өгөгдөл нэмэгдэх бүрт calculate_old_dob() функц ажиллахыг тодорхойлсон байна. Ингэхдээ Expression -ны keyup() /гарнаас товч дараад, авах үед ажиллах/ эвентийн функцийг ашигласан байна.

2 - Without Event хэсэгт процесс дуудагдах үед шууд ажиллах зүйлсийг тодорхойлсон байна. Энэ нь dated /Бүртгэгдсэн огноо/, dateOfBirth /Төрсөн огноо/, age /Нас/ гэсэн path - уудтай утга оруулах талбаруудыг disable() функцын тусламжтай идэвхигүй болгож мөн dated path -тай талбарт тухайн өдрийн огноог буцаах getCurrentDate() функцаар утга оноосон байна. 

3 - Variable & Function хэсэгт calculate_old_dob(), getCurrentDate() зэрэг функцуудад ашиглагдах хувьсагчид болон бусад шалгах үйлдэл хийж буй олон төрлийн функцуудыг оруулсан байна.

Дээрхи байдлаар тохируулж өгсөнөөр ажилтан нэмэх процессын гүйцэтгэл хийгээд үйл ажиллагаа сайжирна. Гэвч Expression -нь үүнээс гадна маш өргөн боломж, томоохон цар хүрээтэй ойлголт юм. Дараах зургаас тохируулгын буюу Expression хэрхэн ажиллассан байгааг харна уу.

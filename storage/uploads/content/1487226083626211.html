Change event гэж юу вэ?

    
    Change event гэдэг нь талбарын утгыг өөрчлөхөд ашигладаг. Тухайн талбарын утга өөрчлөхөд хийгдэх үйлдлүүдийг дотор нь бичиж болно.
Change event нь хулгана шилжих болон энтер дарах үед ажилладаг.

    
    Change event нь Full Expression-н With Event хэсэгт дараах бичиглэлтэй байна.
    
      Change бичиглэл 
[bookTypeId].change(){
  if([bookTypeId].val() == 1){
     [bookNumber] = 1;
  } else {
     [bookNumber] = 3;
  }
};

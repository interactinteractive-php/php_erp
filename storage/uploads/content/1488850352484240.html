SQL CONSTRAINTS

    
    Энэ нь хүснэгт дэх өгөгдөл ямар хэлбэр, форматтай байх зэрэг дүрмийг тодорхойлоход ашигладаг SQL хэлний арга хэрэгсэл буюу хязгаарлалт юм. Баганы болон хүснэгтийн түвшинд хязгаарлалтыг хэрэгжүүлдэг.

Хэрэглээ: Хүснэгтийг анх үүсгэхдээ Constraint ашиглан тухайн баганы өгөгдөл ямар дүрмийн дагуу орохыг тодорхойлж өгдөг бөгөөд зөв харагдацтай хүснэгттэй болох юм. Энэ нь хайлт болон эрэмбэлэлт хийх зэрэг бусад командыг хэрэгжүүлэхэд илүү үр дүнтэй болно.  
  • NOT NULL - Тухайн багана хоосон утга авахгүй гэдгийг илтгэдэг.
  • UNIQUE - Тухайн баганы утгууд өөр хоорондоо ялгаатай байх ёстойг илтгэдэг.
  • PRIMARY KEY - Хүснэгтийн бүх бичлэг хоорондоо давтагдашгүй түлхүүрээр ялгаатай байхыг илтгэдэг.
  • FOREIGN KEY - Өөр хүснэгтийн мөр/бичлэг дэх давтагдашгүй тодорхойлогч
  • CHECK - Тухайн багана дах бүх утгууд нь тодорхой нөхцөлийн дагуу орно гэдгийг илтгэдэг.
  • DEFAULT - Тухайн баганад өгөгдөл оруулаагүй буюу хоосон утга орох үед ямар утга авах ёстойг илтгэдэг.
  • INDEX - Өгөгдлийн сангаас өгөгдлүүдийг маш хурдан татаж авах, харуулах боломж олгодог.
Бичигдэх хэлбэр /syntax/:

CREATE TABLE table_name (
    column1 datatype constraint,
    column2 datatype constraint,
    column3 datatype constraint,
    ....
);

File гэж юу вэ?

    
   

File гэдэг нь Field метадатаны төрөл юм. File төрөлтэй Field нь файл авах зориулалттай талбар юм. File нь зураг, document, text зэрэг олон төрлийн файлын өргөтгөлтэй байж болдог.

    
    File-г үүсгэхдээ Enterprice innovation studio хэсэгт нэмэх товчны тусламжтайгаар мета нэмэх ба баруун талд байрлах төрөл хэсгээс Field-ийг сонгосны дараа өгөгдлийн төрөл хэсгээс File-ийг сонгож үүсгэнэ.

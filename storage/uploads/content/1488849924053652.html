SQL BETWEEN

    
    Энэ нь өгөгдлийн сангийн хүснэгтүүдээс тодорхой нөхцөл, шалгуурын дагуу шүүлт хийхэд ашигладаг WHERE clause-тай зохицож ажилладаг SQL хэлний оператор юм. BETWEEN оператор нь тодорхой утгын хооронд буюу интервал завсарт шүүлт хийх боломж олгодог. Тодорхой утга гэдэгт тоо, тэмдэгт мөр болон он цаг төрлийн утгууд багтдаг.

Хэрэглээ: Өгөгдлийн сангийн хүснэгтүүдээс шүүлт хийхдээ тодорхой утгын хооронд хайлт хийх шаардлага тулгардаг. Тухайлбал 2016 оны 1 -р сараас 3 -р сар хүртэлх нийт захиалгын мэдээллийг харахыг хүсвэл BETWEEN операторыг ашиглах нь тохиромжтой. Хүрээ хязгаарыг тодорхойлохдоо эхлэл болон төгсгөлийн утгыг бичиж өгдөг. Мөн NOT буюу үгүйсгэлийн оператортай нийлж интервалд орохгүй бусад утгуудыг харуулах боломжтой юм.

Бичигдэх хэлбэр /syntax/:
SELECT column_name(s) FROM table_name WHERE column_name BETWEEN / NOT BETWEEN value1 AND value2;
    
    Жишээ-1: Захиалгын жагсаалтаас 1996/07/04 - 1996/07/09/ хоорондох буюу тодорхой хугацааны интервалд шүүлт хийх шаардлагатай болсон гэж үзье. Энэ тохиолдолд Between операторыг ашиглан дараах байдлаар SELECT query бичнэ. 

SELECT *  FROM Customers WHERE Country IN ('Germany', 'France', 'UK');
    
    Зураг 01 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.


Зураг 01 Germany, France, UK улсын хэрэглэгчдийн жагсаалт		    
   
Зурагт IN операторыг хэрэглэсний дараах үр дүнг харуулсан бөгөөд Customers хүснэгтээс Country баганы утга Germany, France, UK зэрэг утгатай байгаа бүх бичлэгүүдийг select хийсэн байна. Энд IN оператор нь OR операторын зарчимаар ажилладаг нь харагдаж байна.

Зураг 03 -с жишээ хоёрт ашиглагдах Suppliers хүснэгтийг харна уу.
    
   
Зураг 03 Suppliers хүснэгт		    
    Жишээ-2: Хэрэглэгчдийн мэдээллийн жагсаалтаас нийлүүлэгч орны хэрэглэгчид байгаа эсэхийг харах шаардлагатай болсон гэж үзье. Энэ тохиолдолд Suppliers хүснэгтийн Country талбарын утгуудыг хайлтын утга болгож дараах байдлаар SELECT query бичнэ.

 SELECT * FROM Customers WHERE Country IN (SELECT Country FROM Suppliers);

     Зураг 04 UK болон USA улсын нийлүүлэгчидтэй ижил орны хэрэглэгчдийн мэдээлэл .    
    Зурагт SELECT query -с гарч ирсэн жагсаалтаас IN операторыг ашиглан хэрхэн шүүлт хийж болохыг харуулсан байна. Өөрөөр хэлбэл Customers хүснэгтийн Country талбарын утга Suppliers хүснэгтийн Country баганы утгуудтай тэнцэж байх үеийн бүх бичлэгийг select хийсэн  байна.

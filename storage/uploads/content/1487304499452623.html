Package гэж юу вэ?

    
    Package нь Veritech ERP | Enterprice innovation studio нэг хуудсанд 2 болон түүнээс дээш тооны дэд цонх үүсгэх боломж олгодог нэг төрлийн метадата юм. Package -н үндсэн хамтарч ажилладаг мета бол меню юм.  Тэдгээр дэд хуудасуудыг харуулах үндсэн 2 төрөлтэй. Үүнд:
  • Tab - меню бүрээр тусдаа дэд цонх үүсгэдэг.
  • One Page - меню бүрийг нэг хуудсанд байхаар үүсгэдэг.
Package үүсгэхдээ Enterprice innovation studio хэсэгт байрлах нэмэх товчны тусламжтайгаар мета нэмэх ба баруун талд байрлах төрөл хэсгээс Package -г сонгож, харагдах загварыг сонгож үүсгэнэ.  
    
   
Зураг 01 Package үүсгэх
    
                                                                                                                                                                                                                                        Зураг 01 Package үүсгэх    
    Package нь меню метадатаг өөрийн элемент болгож авдаг. Харин меню мета нь дуудаж ажиллуулах DataView, Layout, Statement зэрэг метадатаг өөртөө тодорхойлж өгсөн байх шаардлагатай. Package -д менюг дэлгэцийн зүүн талд байрлах нэмэх товчны тусламжтай үзүүлэлт оруулах цонхноос сонгож оруулдаг. 

Жишээ 01: Ажилтануудын бүртгэл мэдээллийг хураангуй болон дэлгэрэнгүй хэлбэрээр харах шаардлагатай болсон гэж үзье. Тэгвэл package ашиглан харуулж болно. Өөрөөр хэлбэл нэг зүйлийг олон төрлийн загвар хэлбэрээр харах шаардлагатай үед ашиглах нь тохиромжтой байдаг. Хоорондоо холбоо хамааралгүй зүйлсийг багцалж харуулах бол Workspace метадата ашиглах нь зохимжтой. 

Зураг 02 Package тохируулах /Ажилтны бүртгэлээр/
    
   

Зураг 02 Package  тохируулах /Ажилтаны бүртгэлээр/		    
   
Дээрхи зурагт ажилтануудын жагсаалтыг товч болон дэлгэрэнгүй хэлбэрээр харуулах Package мета угсарч байгааг харуулсан бөгөөд дэлгэцэнд дүрслэх хэлбэрээр Tab төрлийг сонгосон байна. Харин дэлгэцийн зүүн талд байрлах хэсэгт товч болон дэлгэрэнгүй жагсаалт дуудаж харуулах меню метадатануудыг нэмсэн байна.

Ажилтануудын бүртгэлийг товч болон дэлгэрэнгүй хэлбэрээр харуулах Package метадата тохируулсаны дараах үр дүнг харцгаая.

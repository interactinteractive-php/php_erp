SQL CREATE INDEX

    
    Энэ нь хүснэгтэд индекс үүсгэх зориулалтаар ашиглагддаг SQL хэлний команд юм. Индекс гэдэг нь тухайн хүснэгтээс бичлэг шүүж харах үйлдлийг илүү хурдан болгох зорилготой үйл ажиллагаа юм. 

Хэрэглээ: Маш их хэмжээний өгөгдөлтэй хүснэгтүүдтэй ажиллах үед тэдгээрийг харах хүсэлтийн тооцоолол удаан хийгддэг. Тэдгээр хүснэгтүүдэд индекс үүсгэснээр илүү хурдан харах боломж бүрдэнэ. Индекс нь ерөнхийдөө бол хүснэгтийн бичлэгүүдийг хайлтын алгоритм эсвэл эрэмбэ хэлбэрийн тодорхой бүтцэд оруулдаг юм. 

Бичигдэх хэлбэр /Syntax/:

Тухайн баганы утгууд давхардах үед ашиглах бичиглэл
CREATE INDEX index_name
ON table_name (column1, column2, ...);

Тухайн баганы утгууд ялгаатай байх үед Unique index үүсгэх бичиглэл
CREATE UNIQUE INDEX index_name
ON table_name (column1, column2, ...);


Жишээ-1: Persons хүснэгтийг LastName баганаар индекс үүсгэх шаардлага гарсан гэж үзвэл харгалзах query-г дараах байдлаар бичнэ. Энд тухайн үүсгэж буй индексээ idx угтвартай нэрлэх хэрэгтэй байдаг. Индекст нэр олгосноор буцааж устгахад хэрэг болдог.  

CREATE INDEX idx_lastName ON Persons (LastName);

Жишээ-2: Индекстэй ажиллах query нь өгөгдлийн сангийн удирдах систем бүрт харилцан адилгүй байдаг. Тэгвэл үүсгэсэн индексээ устгахын тулд DROP query -г хэрхэн бичихийг харцгаая.

MS Access:
DROP INDEX index_name ON table_name;

SQL Server:
DROP INDEX table_name.index_name;

DB2/Oracle:
DROP INDEX index_name;

MySQL:
ALTER TABLE table_name
DROP INDEX index_name;

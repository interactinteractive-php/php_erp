SQL IN

    
    Энэ нь өгөгдлийн сангийн хүснэгтүүдээс тодорхой нөхцөл, шалгуурын дагуу шүүлт хийхэд ашигладаг WHERE clause-тай зохицож ажилладаг SQL хэлний оператор юм. IN оператор нь олон утгуудыг тодорхойлж, шүүлт хийх боломж олгодог. 

Хэрэглээ: Өгөгдлийн сангийн хүснэгтүүдээс олон нөхцөл шалгуураар шүүлт хийхдээ бид AND, OR зэрэг операторуудыг ашигладаг. Харин IN оператор нь олон удаа OR тодорхойлж бичсэний товч хэлбэр нь болж өгдөг. Түүнчлэн хамгийн том давуу тал нь тухайн SELECT query ажилласны үр дүнгээс давхар шүүлт хийх боломжийг олгодог юм. 

Бичигдэх хэлбэр /syntax/:
SELECT column_name(s) FROM table_name WHERE column_name IN (value1, value2, ...);

 SELECT column_name(s) FROM table_name WHERE column_name IN (SELECT STATEMENT);

Зураг 01 -с жишээнд ашиглагдах Customers хүснэгтийг харна уу.

    
                                                                                                                                        Зураг 01 Customers хүснэгт    
   
Жишээ-1: Хэрэглэгчийн мэдээллийн жагсаалтаас Germany, France, UK улсын хэрэглэгчдийн мэдээллийг харах шаардлагатай болсон гэж үзье. IN операторыг ашиглан дараах байдлаар SELECT query бичнэ. 

SELECT *  FROM Customers WHERE Country IN ('Germany', 'France', 'UK');

Зураг 02 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.
    
   

Зураг 02 Germany, France, UK улсын хэрэглэгчдийн жагсаалт		    
   
Зурагт IN операторыг хэрэглэсний дараах үр дүнг харуулсан бөгөөд Customers хүснэгтээс Country баганы утга Germany, France, UK зэрэг утгатай байгаа бүх бичлэгүүдийг select хийсэн байна. Энд IN оператор нь OR операторын зарчимаар ажилладаг нь харагдаж байна.

Зураг 03 -с жишээ хоёрт ашиглагдах Suppliers хүснэгтийг харна уу.
    
   
Зураг 03 Suppliers хүснэгт		    
    Жишээ-2: Хэрэглэгчдийн мэдээллийн жагсаалтаас нийлүүлэгч орны хэрэглэгчид байгаа эсэхийг харах шаардлагатай болсон гэж үзье. Энэ тохиолдолд Suppliers хүснэгтийн Country талбарын утгуудыг хайлтын утга болгож дараах байдлаар SELECT query бичнэ.

 SELECT * FROM Customers WHERE Country IN (SELECT Country FROM Suppliers);

Зураг 04 -с дээрх SELECT query хэрхэн ажилласаныг харна уу.
    
     Зураг 04 UK болон USA улсын нийлүүлэгчидтэй ижил орны хэрэглэгчдийн мэдээлэл .    
    Зурагт SELECT query -с гарч ирсэн жагсаалтаас IN операторыг ашиглан хэрхэн шүүлт хийж болохыг харуулсан байна. Өөрөөр хэлбэл Customers хүснэгтийн Country талбарын утга Suppliers хүснэгтийн Country баганы утгуудтай тэнцэж байх үеийн бүх бичлэгийг select хийсэн  байна.

SQL DATE FUNCTIONS

    
    SQL -д анхнаас суулгаж өгсөн цаг хугацаа төрлийн өгөгдөлд тооцоолол хийх зориулалттайгаар ашигладаг олон төрлийн функцууд байдаг. Дараах хүснэгтээс харна уу.

MySQL -н Date функцууд
Функц Тайлбар
NOW() Тухайн өдрийн огноо болон цагийг хамтад нь буцаана
CURDATE() Тухайн өдрийн огноог буцаана
CURTIME() Тухайн өдрийн цагийг буцаана
DATE() Цаг хугацаа болон огноо төрлийн нийлмэл өгөгдлөөс зөвхөн огноо хэсгийг салгаж буцаана
EXTRACT() Огноо эсвэл цаг хугацаа төрлийн өгөгдөөс нэгж хэсгийг нь салгаж буцаана
DATE_ADD() Тодорхой цаг хугацааны интервалын нийлбэрийг олж буцаана
DATE_SUB() Тодорхой цаг хугацааны интервалын ялгаврыг олж буцаана
DATEDIFF() 2 цаг хугацаны зөрүүг буцаана
DATE_FORMAT() Цаг хугацаа төрлийн өгөгдлүүдийг олон өөр форматаар харуулдаг

SQL Server -н Date функцууд
Функц Тайлбар

DATEPART() Огноо эсвэл цаг хугацаа төрлийн өгөгдөөс нэгж хэсгийг нь салгаж буцаана
DATEADD() Тодорхой цаг хугацааны интервалын нийлбэр болон ялгаврыг олж буцаана
DATEDIFF() 2 цаг хугацаны зөрүүг буцаана
CONVERT() Цаг хугацаа төрлийн өгөгдлүүдийг олон өөр форматаар харуулдаг

FORMAT() Талбар ямар форматаар дүрслэгдэхийг харуулна
NOW() Системийн тухайн цаг хугацаа, огноог буцаана

SQL Date Data Types буюу бичиглэл формат

MySQL дараах цог хугацаа, огноог илэрхийлэх өгөгдлийн төрлүүдтэй бөгөөд өгөгдлийн санд бичигдэх форматын дагуу хадгалагддаг.

  • DATE - format YYYY-MM-DD
  • DATETIME - format: YYYY-MM-DD HH:MI:SS
  • TIMESTAMP - format: YYYY-MM-DD HH:MI:SS
  • YEAR - format: YYYY or YY

SQL Server дараах цог хугацаа, огноог илэрхийлэх өгөгдлийн төрлүүдтэй бөгөөд өгөгдлийн санд бичигдэх форматын дагуу хадгалагддаг.

  • DATE - format YYYY-MM-DD
  • DATETIME - format: YYYY-MM-DD HH:MI:SS
  • SMALLDATETIME - format: YYYY-MM-DD HH:MI:SS
  • TIMESTAMP - format: a unique number
Жишээ-1: Дараах query нь огноо төрлөөр шүүлт хийж байна. Энд огноо болон цаг хугацаа төрлийн өгөгдлийн харьцуулалт шүүж хийхдээ харгалзах утгыг дан эсвэл давхар хашилтанд хийж өгдөг.

SELECT * FROM Orders WHERE OrderDate='2008-11-11';

Дээрх select query нь Orders хүснэгтээс OrderDate буюу захиалгын огноо талбар нь 2008-11-11 ны өдөр хийгдсэн бүх захиалгуудыг харуулах юм.

SQL SELECT

    
    Энэ нь өгөгдлийн сантай харьцах үед хамгийн их ашиглагддаг SQL хэлний statement юм. SELECT - г өгөгдлийн сангийн хүснэгтүүдээс бичлэг буюу өгөгдлүүдийг сонгож харуулах зорилгоор ашигладаг. Түүнчлэн SQL хэлний бусад statements - тэй зохицон ажиллах бүрэн боломжтой.

Хэрэглээ: Системтэй харьцан ажиллаж байгаа үед өгөгдлийн санд хадгалж буй өгөгдлүүдээ харж, дүн шинжилгээ хийх шаардлага бидэнд зайлшгүй тулгардаг. Энэ үед SELECT query бичиж, өөрийн харахыг хүссэн үр дүнг хүснэгт байдлаар харах боломжтой болно. 

Бичигдэх хэлбэр /syntax/:
Хүснэгтээс өөрийн харахыг хүссэн баганыг сонгон харуулах бол дараах хэлбэрээр бичнэ.
SELECT column_name1, column_name2 ... FROM table_name;

Хүснэгтийн бүх талбарын өгөгдлүүдийг харуулах бол дараах хэлбэрээр бичнэ.
SELECT * FROM table_name;

Жишээ-1: Хэрэглэгчдийн хаяг байршилын мэдээллийг агуулсан жагсаалт харуулах шаардлагатай болсон гэж үзье.

SELECT * FROM Customers;

Зураг 01 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.
    
     
                                                                                                                                                                                                                            Зураг 01 хэрэглэгчдийн жагсаалт

    
    Зурагт өгөгдлийн санд байрлах Customers хүснэгтийн мэдээлийг SELECT query ашиглан харуулсан байна. Энд * тэмдэгт нь тухайн хүснэгтийн бүх талбарын select хийж байгааг илэрхийлнэ.

Жишээ-2: Customers хүснэгтээс хэрэглэгчийн нэр болон амьдардаг хотын мэдээллийг агуулсан жагсаалт харуулах шаардлагатай болсон гэж үзье.

SELECT CustomerName, City FROM Customers;

Зураг 02 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.
    
     
                                                                                                               Зураг 02 Customers хүснэгтээс хэрэглэгчийн нэр, хот гэсэн багануудыг сонгож Select хийсэн үр дүн		    
   
Зурагт Customers хүснэгтээс харах шаардлагатай багануудын нэрийг таслалаар тусгаарлан тодорхойлж, select хийж байгааг харуулсан байна.

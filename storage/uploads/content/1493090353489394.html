SQL DROP TABLE

    
    Энэ нь ӨСУС -д өгөгдлийн санд байрлах хүснэгтийг устгах SQL хэлний statement буюу команд юм.

Хэрэглээ: Хүснэгт устгахдаа маш анхааралтай байх шаардлагатай бөгөөд ихэнхи тохиолдолд хүснэгтийг буруу үүсгэсэн илүү буюу давхар үүсгэсэн үед ашигладаг. Хэрэв анхаарал алдвал өгөгдлийн сангийн хэвийн үйл ажиллагаанд саад учируулж асуудалд орох өндөр магадлалтай.

Бичигдэх хэлбэр /syntax/:
Богино хэлбэр
DROP TABLE tablename;

Жишээ-1: Өгөгдлийн санг дараах байдлаар устгана.  

DROP DATABASE testDB;

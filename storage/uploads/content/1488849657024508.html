SQL ORDER BY

    
    Энэ нь өгөгдлийн сангийн хүснэгтүүдээс өгөгдлийг өсөхөөр болон буурахаар эрэмбэлэн харуулах зорилгоор ашиглагддаг SQL хэлний түлхүүр үг юм. ORDER BY -ийг өсөх болон буурахаар тодорхойлж өгөөгүй бол шууд өсөхөөр эрэмбэлэгддэг. Хэрэв буурахаар эрэмбэлэн харуулахийг хүсвэл DESC түлхүүрийг хэрэглэнэ.

Хэрэглээ: Ерөнхийд нь ORDER BY -г бид номын агуулга хэсгээс харж болно. Номын агуулгад гарчигийг нь эрэмбэлэгдсэн байдгийг санаж байгаа байх. Тэр нь бидэнд хаанаас, юу хайх вэ гэдгийг харахад хялбархан болгодог. Түүнтэй адил өгөгдлийн санд байгаа хүснэгт дахь бичлэгийг эрэмбэлснээр хайхад хялбар болно.

Бичигдэх хэлбэр /syntax/:

SELECT column_name1, column_name2 ... FROM table_name ORDER BY column_name1, column_name2 ... ASC|DESC;


Зураг 01-с  жишээнд ашиглах хүснэгтийг харна уу.
    
     
  
                                                                                                                                                                                                                                   Зураг 01  Customers хүснэгт

    
    Жишээ-1: "Customers" хүснэгтийн бүх хэрэглэгчийг нэрээр нь эрэмбэлэн харуулах хэрэгтэй боллоо гэж үзье. Энэ тохиолдолд ORDER BY ашиглана.

SELECT * FROM Customers ORDER BY CustomerName;

Зураг 02 -с дээрх ORDER BY түлхүүрийн хамт тодорхойлогдсон SELECT query хэрхэн ажилласан байгааг харна уу.

    
   
                                                                                                                                                                                     Зураг 02 CustomerName -ийг өсөхөөр эрэмблэсэн байна

    
    Зурагт ORDER BY түлхүүрийн хэрэглээг харуулсан бөгөөд Customers хүснэгтийг CustomerName баганаар эрэмбэлсэн байна. Хэрэв өсөхөөр эсвэл буурахаар эрэмблэх бол ASC болон DESC түлхүүр үгүүдийг ашиглаж болно.

Жишээ-2: Customers хүснэгтийн бүх хэрэглэгчийн мэдээллийг DESC түлхүүрийг ашиглан буурахаар эрэмбэлэн харуулах шаардлага гарсан гэж үзье.

SELECT *FROM Customers ORDER BY CustomersName DESC;

Зураг 03 -с DESC түлхүүр ашигласнаар ямар ялгаа гарсаныг харна уу.

    
                                                                                                                                                                                 Зураг 03 CustomerName баганыг буурахаар эрэмблэсэн байна

    
    Зурагт ORDER BY түлхүүр үгийн хэрэглээг харуулсан бөгөөд Customers хүснэгтийг CustomerName баганаар буурахаар эрэмбэлсэн байна. Хэрэв өсөхөөр эсвэл буурахаар эрэмблэх бол ASC болон DESC түлхүүр үгүүдийг ашиглаж болно.

Жишээ-3: Customers хүснэгтийн Country баганыг ASC ашиглан өсөхөөр, Address баганыг буурахаар эрэмбэлж үзье. Энд нэгээс олон баганаар эрэмбэлэлт хийж болохыг харуулах юм.

SELECT *FROM Customers ORDER BY Country ASC,  Address DESC;

Зураг 04 -с ASC|DESC -ийг ашиглан эрэмбэлэж болно гэдгийг харна уу.
    
                                                                                                                                                                                                                                                                                                Зураг 04 Country -ийг буурахаар Address -ийг өсөхөөр эрэмблэсэн байна

    
    Зурагт ASC|DESC түлхүүрийг хэрэглэн улсыг өсөхөөр, хаягийг буурахаар эрэмбэлэхээр зааж өгсөн байна. Гэвч бидний харж байгаагаар хаягийн утгууд нь өөрчлөгдөөгүй байна. Үүний шалтгаан нь эхлээд бид улсийг өсөхөөр өгсөн тул улс нь эрэмблэгдсэн. Харин Address -аар эсрэгээр нь эрэмбэлнэ гэдэг нь өгөгдлийн сангийн бичлэгийг өөрчлөх хэрэгтэй болно. Ингэж болохгүй учир Address өөрчлөгдөөгүй.

Code гэж юу вэ?

    
   

Code гэдэг нь Field метадатаны төрөл юм. Code төрөлтэй Field нь тухайн хүснэгтээс зөвхөн код илэрхийлэх утгыг авах зориулалттай талбар юм. 

    
    Code-г үүсгэхдээ Enterprice innovation studio хэсэгт нэмэх товчны тусламжтайгаар мета нэмэх ба баруун талд байрлах төрөл хэсгээс Field-ийг сонгосны дараа өгөгдлийн төрөл хэсгээс Code-ийг сонгож үүсгэнэ.

Зураг 01 Code үүсгэх
   
    Зураг 01 Code үүсгэх    
     
Жишээ нь: Ерөнхий дөө бол Code талбарыг нэг удаа тодорхойлдог бөгөөд тухайн хүснэгтийн код илэрхийлэх талбарын утгыг харуулах үед Code төрлийн Field metadata үүсгэж ашигладаг. Код гэдэгт ажилтаны код, бүтээгдэхүүний код, үзүүлэлтийн код зэрэг хамаарагдана.

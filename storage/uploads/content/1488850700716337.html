SQL COMMENTS

    
    Энэ нь SQL илэрхийлэллийг хэсэгчлэн тайлбарлаж бичих боломж олгодог арга хэрэгсэл юм. Comment болгосон хэсгийн бичиглэл нь ажиллагаанд нөлөөлдөггүй.

Хэрэглээ: Бүхий програмын кодчилолд тайлбарыг ашигладаг. Учир нь тухай бичиж буй кодын хэсгийг өөр хүн эсвэл удаан хугацааны дараа эргэн хараад ойлгоход хүндрэлтэй байдаг. Хэрэв өөрөө бичиж хадгалаагүй л бол ойлгоход хэцүү. Тиймээс тайлбар төдий чинээ бичнэ ойлгомжтой байдал нэмэгдэж кодчилох техник сайжирна гэдэг бичигдээгүй хууль бий. Үүнтэй ижил томоохон query бичиж байгаа үед comment ашиглан sql илэрхийлэлээ хэсэгчлэн тайлбарлаж явах нь үр дүнтэй юм. 

Бичигдэх хэлбэр /syntax/:
Single line comment
--Select all

MultiLine comment
/* Select all the columns
of all the records
in the Customers table: */

Жишээ-1: Single line comment -г sql илэрхийллийн эхлэл болон төгсгөл хэсэгт дараах байдлаар ашигладаг. 
--Customers хүснэгтийн бүх бичлэгийг харуулж байна
SELECT * FROM Customers;

SELECT * FROM Customers -- WHERE City='Berlin';

Жишээ-2: MultiLine comment хийх аргачлалыг sql илэрхийллийн аль ч хэсэгт ашиглаж болох давуу талтай болохыг дараах бичиглэлүүдээс харна уу.

SELECT CustomerName, /*City,*/ Country FROM Customers;

SELECT * FROM Customers WHERE (CustomerName LIKE 'L%'
OR CustomerName LIKE 'R%' /*OR CustomerName LIKE 'S%'
OR CustomerName LIKE 'T%'*/ OR CustomerName LIKE 'W%')
AND Country='USA'
ORDER BY CustomerName;

Ингэж бичих нь sql query илэрхийллүүдээ ахин дахин бичиж ажиллуулахаас сэргийлдэг.

SQL DROP DATABASE

    
    Энэ нь ӨСУС -д өгөгдлийн сан устгах SQL хэлний statement буюу команд юм.

Хэрэглээ: Өгөгдлийн сан устгахдаа маш анхааралтай байх шаардлагатай бөгөөд ихэнхи тохиолдолд өгөгдлийн сан илүү буюу давхар үүсгэсэн үед ашигладаг. Хэрэв анхаарал алдвал бүхэл бүтэн байгууллагын өгөгдлийн санг устгаж, асуудалд орох өндөр магадлалтай.

Бичигдэх хэлбэр /syntax/:
Богино хэлбэр
DROP DATABASE databasename;

Жишээ-1: Өгөгдлийн санг дараах байдлаар устгана.  

DROP DATABASE testDB;

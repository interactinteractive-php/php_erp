SQL Логик операторууд
Оператор Тайлбар
ALL Хэрэв бүх дэд query нөхцөлийг хангаж байвал үнэн байна
AND Хэрэв бүх нөхцөл тус бүртээ биелэх бол үнэн байна
ANY Хэрэв аль нэг дэд query нөхцөлийг хангаж байвал үнэн байна
BETWEEN Хэрэв утга тодорхой хүрээнд байх бол үнэн байна
EXISTS Хэрэв дэд query нэг эсвэл түүнээс дээш тооны бичлэг буцаавал үнэн байна
IN Хэрэв утга жагсаалтын аль нэг утгатай тэнцүү бол үнэн байна
LIKE Хэрэв хайлтын утгад харгалзах төсөөтэй утга олдвол үнэн байна
NOT Хэрэв нөхцөлтэй таарахгүй байвал үнэн байна
OR Хэрэв нөхцөлүүдийн аль нэг биелэж байвал үнэн байна
SOME Хэрэв зарим дэд query нөхцөлийг хангаж байвал үнэн байна

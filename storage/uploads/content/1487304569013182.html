Дээрхи зурагт харуулсан тохиргоонуудын тайлбарыг дараах жагсаалтаас харна уу.
  • Процесс - Dashboard Card нь хадгалах үед процесс төрөлтэй болж үүсдэг.  Өөрийн үзүүлэлтийн нэр , кодыг процесс тохиргоо авдаг учир тохируулах шаардлагагүй. 
  • Текст - Энэ нь карт дээр харагдах нэршилийг оруулах тохиргоо бөгөөд маш тодорхой буюу оновчтой тохируулах хэрэгтэй.
  • Текст зэрэгцүүлэлт - Картны аль талд текст байрлах эсэхийг тохируулдаг. Стандарт утгатай тохируулагдсан байдаг.
  • Линк - Энгийн вэб URL оруулахгүй бөгөөд карт дээр дарахад дуудагдах жагсаалтын үзүүлэлтийн дугаарыг тохируулдаг. Ингэснээр тухайн картны илэрхийлж буй жагсаалтыг харах боломжтой болох юм.
  • Фонтын хэмжээ - Үсгийн хэмжээг тохируулна. Стандарт утгатай тохируулагдсан байдаг.
  • Өргөн, Өндөр - Картны дүрслэгдэх хэмжээг тохируулна. Стандарт утгаар тохируулагдсан байдаг.
  • Суурь өнгө - Ерөнхийдээ дэвсгэр өнгөний тохиргоог оруулдаг. Хянах самбарт олон карт байрлах учир тэдгээрийн өнгөний зохилдлогоог бодож өнгийг сонгодог. Бусад модулийн хяналтын самбарын жишиг өнгийг харж оруулвал зохимжтой.
  • Font icon - Картанд зөвхөн дан өнгө тохируулахаас гадна дүрс зураг оруулдаг. Энэ нь илэрхийлж буй жагсаалттай холбоо хамааралтай байх хэрэгтэй.
  • Data View - Тухайн картанд тооцоолж харуулах жагсаалтыг тохируулна.
  • Column name - Тухайн тохируулагдсан жагсаалтын аль багана дээр тооцоолол хийхийг тохируулна.
  • Aggregate name - Тус тохиргоо нь сонгогдсон баганын утгуудыг ашиглан ямар функцаар тооцоолол хийхийг тохируулдаг. Функц гэдэгт агрегат буюу дараах функцууд багтдаг.

Функц Тайлбар
Тоолох /Count/ Жагсаалтын бичлэгүүдийн тоог тоолдог. Null утга авдаггүй баганыг тохируулах шаардлагатай
Нийлбэр /Sum/ Тухайн баганы утгуудын нийлбэр дүнг тооцоолдог. Тоон утга бүхий баганыг тохируулах шаардлагатай.
Дундаж /Average/ Тухайн баганы утгуудын дундаж дүнг тооцоолдог. Тоон утга бүхий баганыг тохируулах шаардлагатай. 
Хамгийн их /Max/ Тухайн баганы хамгийн их утгыг олж харуулна.
Хамгийн бага /Min/ Тухайн баганы хамгийн бага утгыг олж харуулна.
Сүүлчийн утга /Last/ Тухайн баганы хамгийн сүүлчийн утгыг харуулна.

 
Жишээ 01: Энэ сард төрсөн ажилтаны тоог харуулах Dashboard Card үүсгэе. Ингэхдээ тухайн сард төрсөн ажилтануудын жагсаалтыг үүсгэсэн байх шаардлагатай.

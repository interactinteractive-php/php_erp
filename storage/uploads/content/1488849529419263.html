SQL WHERE

    
    Энэ нь өгөгдлийн сангийн хүснэгтүүдээс тодорхой нөхцөл, шалгуурын дагуу шүүлт хийхэд ашигладаг SQL хэлний Clause /гишүүн, бүлэг, заалт/ юм. UPDATE, DELETE, SELECT зэрэг query - тэй зохицож ажилладаг бөгөөд шүүлт хийх нөхцөл өө тодорхойлох доо дараах төрлийн операторуудын тусламжтай бичнэ.
Оператор Тайлбар
= Тэнцүү буюу нөхцөл нь гарцаагүй таарч байх үед биелнэ.
<> Тэнцүү биш буюу ялгаатай утгатай байх үед биелнэ. Зарим SQL хувилбаруудад != байдалтай бичигдэнэ.
> Нөхцөлд тодорхойлсон утга баганы утгаас их байх үед биелнэ.
< Нөхцөлд тодорхойлсон утга баганы утгаас бага байх үед биелнэ.
>= Нөхцөлд тодорхойлсон утга баганы утгаас их юмуу тэнцүү байх үед биелнэ.
<= Нөхцөлд тодорхойлсон утга баганы утгаас бага юмуу тэнцүү байх үед биелнэ.
BETWEEN Баганы утга нь Нөхцөлд тодорхойлсон утгуудын хооронд байх үед биелнэ.
LIKE Нөхцөлд тодорхойлсон утга баганы утгатай төстэй байх үед биелнэ.
IN Шүүлт хийж буй баганы утга нөхцөлд тодорхойлсон утгуудын аль нэгэнтэй нь тэнцүү байх үед биелнэ.

Хэрэглээ: Өгөгдлийн сангийн хүснэгтүүдтэй харьцан ажиллах үед өөрийн хүссэн үр дүнг шүүлт хийж харахаас гадна алдаатай бичлэгийг засах, устгах зэрэгт WHERE clause  тодорхойлж ашиглана.

Бичигдэх хэлбэр /syntax/:
SELECT column_name1, column_name2 ... FROM table_name WHERE condition;
 
Жишээ-1: Ажилчдын мэдээллийн жагсаалтаас Margaret нэртэй ажилчины мэдээллийг харуулах шаардлагатай болсон гэж үзье.

SELECT * FROM Employees;

Зураг 01 -с дээрх SELECT query хэрхэн ажилласан байгааг харна уу.
    
       
                                                                                                                                                                                                                                    Зураг 01 ажилчдын жагсаалт

    
    Зурагт Employees хүснэгтийн мэдээллийг SELECT query ашиглан харуулсан байна. Энд * тэмдэгт нь тухайн хүснэгтийн бүх талбарын select хийж байгааг илэрхийлнэ.

SELECT * FROM Employees WHERE FirstName = 'Margeret';

Зураг 02 -с дээрх WHERE clause -н хамт тодорхойлогдсон SELECT query хэрхэн ажилласан байгааг харна уу.
    
   
                                                                                                                                                                                                       Зураг 02 Margeret нэртэй ажилчны мэдээлэл		    
   
Зурагт WHERE clause -н хэрэглээг харуулсан бөгөөд Employees хүснэгтээс '=' операторын тусламжтайгаар FirstName баганы утга 'Margaret' -тэй тэнцүү байх үейин бичлэгийг бүх талбараар select хийж харуулсан байна. Энд тэмдэгт мөр /varchar/ болон цаг хугацаа /DateTime/ төрлийн өгөгдлүүдийг нөхцөл болгон бичихдээ дан хашилтан дунд буюу 'data' байдлаар бичнэ. Хэрэв int, float, double зэрэг тоон төрлийн утгаар нөхцөл бичих гэж байгаа бол дан хашилтгүй /EmployeeId = 4/ шууд бичнэ.

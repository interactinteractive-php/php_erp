SQL NOT NULL Constraint

    
    Энэ төрлийн хязгаарлалт нь бичлэг нэмэх болон засахад тухайн талбарын утгыг хоосон утгатайгаар оруулж болохгүйг илтгэдэг.  

Хэрэглээ: Хүснэгтийн анхдагч горимд Null утга оруулж болдог бөгөөд Not Null хязгаарлалтыг ашигласнаар заавал утга оруулах шаардлагатай талбаруудыг null утгатай байхаас сэргийлдэг.  

Бичигдэх хэлбэр /syntax/:

CREATE TABLE table_name (
    column1 datatype constraint,
    column2 datatype constraint,
    column3 datatype constraint,
    ....
);

Жишээ-1: Persons хүснэгтийг үүсгэхдээ хоосон утгатай байж болохгүй багануудад NOT NULL хязгаарлалтыг дараах байдлаар тодорхойлж өгнө.

CREATE TABLE Persons (
    ID int NOT NULL,
    LastName varchar(255) NOT NULL,
    FirstName varchar(255) NOT NULL,
    Age int
);

Дээрх Persons хүснэгтийг үүсгэх query -д ID, LastName, FirstName зэрэг талбарууд null утгатай байж болохгүйг тодорхойлж өгсөн байна.

SQL SELECT TOP, ROMNUM, LIMIT

    
    Энэ нь өгөгдлийн сангийн дэлгэцэнд дүрслэх бичлэгүүдийн тоог хязгаарлаж өгдөг clause буюу гишүүн юм.

Хэрэглээ: Их хэмжээний бичлэгтэй хүснэгтэнд SELECT TOP -ийг ашиглахад тохиромжтой. Их хэмжээний бичлэг буцаах нь дүрслэх болон ажиллах хурданд нөлөөлдөг. Гэвч эдгээрийг бүх өгөгдлийн сан удирдах систем SELECT TOP clause -ийг дэмждэггүй. MySQL хэлэнд LIMIT clause ийг ашиглан хягааралдаг бол Oracle -д ROWNUM -г ашигладаг.

Бичигдэх хэлбэр /syntax/:
SQL Server / MS Access Syntax:

SELECT TOP number | table_name percent column_name(s) FROM table_name WHERE condition;

MySQL Syntax:

SELECT column_name(s)  FROM table_name WHERE condition LIMIT number;

Oracle Syntax:

SELECT column_name(s) FROM table_name WHERE  ROWNUM <= number;

Зураг 01 -с Дараах хүснэгтийг бид жишээнд ашиглана..
    
       Зураг 01 Poducts - Бүтээгдхүүний жагсаалт   
  		    
    Жишээ-1: Prdoduct хүснэгтээс эхний 3 бичлэгийг л авч дэлгэцэнд харуулах шаардлагтай боллоо гэж үзве. Дараах 3 хэдбэрээр ийж болно.

 1. SELECT TOP ашиглаж:
SELECT TOP 3 * FROM Products;

2. LIMIT clause ашиглаж:
SELECT * FROM Products; LIMIT 3;

3. ROWNUM ашиглаж
SELECT * FROM Products; WHERE ROWNUM <= 3;

Зураг 02 -с дээрх query хэрхэн ажилласан байгааг харна уу.		    
     
                                                                                                             Зураг 02 Products хүснэгтээс эхний 3 бичлэг авсан байна.  		    
    PERCENT

Энэ нь SELECT TOP clause -тай цуг хэрэгллэгддэг бөгөөд хязгаарлалтын өөр нэг арга юм. PERCENT нь хүснэгтийн нийт бичлэгүүдийг хувиар илэрхийлж, харгалзах тооны бичлэгүүдийг буцаадаг.

Жишээ-2: Одоо PERCENT ашиглаж Products хүснэгтээс нийт бичлэгийн 30% -тай тэнцэх тооны бичлэгийг дэлгэцэнд харуулж үзье.

SELECT TOP 30 PERCENT * FROM  Products;

.Зураг 03 -с дээрх query хэрхэн ажилласан байгааг харна уу.		    
    Зураг 03 Products хүснэгтээс 30% бичлэгийг авсан байна.     
    WHERE clause ашиглан хязгаарлалтаа илүү нарийн болгож болно. Мөн LIMIT, ROWNUM 2 дээр адилхан ашигдаж болно.

Жишээ-3: Одоо WHERE clause ашиглаж CategoryID нь 2 той тэнцүү эхний 3 бичлэгийг авч үзье. 

1. SELECT TOP ашигласан:
SELECT TOP 3 * FROM Products WHERE CategoryID=2;

2. LIMIT clause ашигласан:
SELECT *  FROM Products WHERE CategoryID=2 LIMIT 3;

3. ROWNUM ашигласан:
SELECT *  FROM Products WHERE CategoryID=2 AND ROWNUM <=3;

SQL DEFAULT Constraint

    
    Энэ нь хүснэгтийн тухайн багана анхдагч утгатай байх боломж олгодог SQL хэлний хязгаарлалтын нэгэн төрөл юм. 

Хэрэглээ: Хэрэв хүснэгтэд шинээр бичлэг нэмэхдээ null утгатайгаар оруулвал Default хязгаарлалт тавьсан баганы утга анхдагч хэлбэрээр нэмэгдэн ордог. 

Жишээ-1: Persons хүснэгтийг үүсгэхдээ City багана анхдагч утгатай байхаар Default хязгаарлалт тавьж дараах байдлаар тодорхойлж бичнэ. Энд хэрэв null утгатай буюу өөр ямар нэг утга тодорхойлоогүй бол хязгаарлалт биелэх юм.

My SQL / SQL Server / Oracle / MS Access:
CREATE TABLE Persons (
    ID int NOT NULL UNIQUE,
    LastName varchar(255) NOT NULL,
    FirstName varchar(255),
    Age int,
    City varchar(255) DEFAULT 'Sandnes'
);

Жишээ-2: Orders хүснэгтийг үүсгэхдээ захиалгын огноо талбарыг Default хязгаарлалтаар тухайн өдрийн огноо авдаг байхаар дараах байдлаар тодорхойлж бичнэ. 

CREATE TABLE Orders (
    ID int NOT NULL,
    OrderNumber int NOT NULL,
    OrderDate date DEFAULT GETDATE()
);

Жишээ-3: Тухайн хүснэгтийг аль хэдийн үүсгэсэн бол ALTER TABLE командаар засахдаа Constraint буюу хязгаарлалт тодорхойлж болно. Persons хүснэгтийг үүсгэсэн гэж үзээд ID талбарыг Unique байх ёстой гэдгийг дараах байдлаар тодорхойлж бичнэ. 

MySQL:
ALTER TABLE Persons ALTER City SET DEFAULT 'Sandnes';

SQL Server / MS Access:
ALTER TABLE Persons ALTER COLUMN City SET DEFAULT 'Sandnes';

Oracle:
ALTER TABLE Persons MODIFY City DEFAULT 'Sandnes';

Жишээ-4: Тухайн үүсгэсэн хязгаарлалт шаардлагагүй гэж үзвэл ALTER TABLE DROP командын тусламжтай устгах бол дараах байдлаар бичнэ.

MySQL:
ALTER TABLE Persons ALTER City DROP DEFAULT;

SQL Server / Oracle / MS Access:
ALTER TABLE Persons ALTER COLUMN City DROP DEFAULT;
